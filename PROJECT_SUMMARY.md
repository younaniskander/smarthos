## 🚀 كيفية الاستخدام السريع

### الخطوة 1: تحميل النموذج
```python
import pickle
with open('movement_model.pkl', 'rb') as f:
    model = pickle.load(f)
with open('feature_scaler.pkl', 'rb') as f:
    scaler = pickle.load(f)
```

### الخطوة 2: استخراج الميزات
```python
# انظر usage_examples.py لرؤية الدالة الكاملة
features = extract_motion_features(patient_data_X, patient_data_y)
```

### الخطوة 3: التنبؤ
```python
features_scaled = scaler.transform(features.reshape(1, -1))
prediction = model.predict(features_scaled)
probability = model.predict_proba(features_scaled)
```

### الخطوة 4: الحصول على النتيجة
```python
if prediction[0] == 0:
    print("✓ حركة طبيعية")
else:
    print("⚠️ حركة غير طبيعية")
```
